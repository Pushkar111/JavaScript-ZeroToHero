Form Events

Triggered during user interaction with form elements.

Examples:

  • input : Fired when the value of an input element changes.[present changes shows]
  • change : Fired when the value of an input, select, or textarea is changed.[after we out elements that changed can be prints or you can say last changes shows]
  • focus : Fired when an element gains focus.
  • blur : Fired when an element loses focus.
  • submit : Fired when a form is submitted.
  • reset : Fired when a form is reset.
